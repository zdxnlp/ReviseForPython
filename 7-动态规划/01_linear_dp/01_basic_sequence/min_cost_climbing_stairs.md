# 使用最小花费爬楼梯

## 题目

给定数组 `cost`，其中 `cost[i]` 表示踩到第 `i` 阶需要付出的体力值。每次可以爬 `1` 阶或 `2` 阶，问到达楼顶的最小花费。

## 为什么是线性 DP

到达某一阶的最优解只和前一阶、前两阶的最优解有关，属于典型的一维最优值递推。

## 四步法

1. 定义状态：`dp[i]` 表示到达第 `i` 阶时的最小花费。
2. 写转移：到 `i` 可以从 `i - 1` 或 `i - 2` 过来，所以 `dp[i] = min(dp[i - 1] + cost[i - 1], dp[i - 2] + cost[i - 2])`。
3. 初始化：`dp[0] = 0`，`dp[1] = 0`，因为可以从第 `0` 阶或第 `1` 阶开始。
4. 遍历顺序：从 `2` 到 `n` 递推，答案是 `dp[n]`。

## 递推阶梯

- 到第 `2` 阶：比较从 `0` 来还是从 `1` 来
- 到第 `3` 阶：比较 `dp[2] + cost[2]` 和 `dp[1] + cost[1]`
- 一路向上维护最小值

关键点是：`dp[i]` 代表已经站到 `i`，而不是踩到 `i - 1`。

## 公式

`dp[i] = min(dp[i - 1] + cost[i - 1], dp[i - 2] + cost[i - 2])`

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`，可优化到 `O(1)`
