# 不同路径

## 题目

机器人位于 `m x n` 网格左上角，每次只能向右或向下走一步，问到右下角共有多少条不同路径。

## 为什么是二维 DP

状态由两个坐标 `(i, j)` 决定，当前位置的答案依赖上方和左方两个二维状态。

## 四步法

1. 定义状态：`dp[i][j]` 表示走到格子 `(i, j)` 的路径数。
2. 写转移：到 `(i, j)` 只能从上方或左方来，所以 `dp[i][j] = dp[i - 1][j] + dp[i][j - 1]`。
3. 初始化：第一行全为 `1`，第一列全为 `1`。
4. 遍历顺序：从上到下、从左到右，答案是 `dp[m - 1][n - 1]`。

## 递推阶梯

- 第一行只能一直向右走
- 第一列只能一直向下走
- 其余格子的路径数等于“上方路径数 + 左方路径数”

## 公式

`dp[i][j] = dp[i - 1][j] + dp[i][j - 1]`

## 复杂度

- 时间复杂度：`O(mn)`
- 空间复杂度：`O(mn)`
