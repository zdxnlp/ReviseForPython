# 回文子串个数

## 题目

给定字符串 `s`，统计其中回文子串的数量。

## 为什么是区间 DP

某个区间 `s[i..j]` 是否为回文，依赖它内部更短区间 `s[i + 1..j - 1]` 是否为回文。

## 四步法

1. 定义状态：`dp[i][j]` 表示子串 `s[i..j]` 是否为回文。
2. 写转移：
   - 若 `s[i] != s[j]`，则不是回文
   - 若 `s[i] == s[j]`，当 `j - i <= 1` 或 `dp[i + 1][j - 1]` 为真时，它是回文
3. 初始化：单个字符一定是回文。
4. 遍历顺序：仍然从短区间推长区间，因此 `i` 要逆序，`j` 要正序。

## 递推阶梯

- 长度为 `1` 的区间全部成立
- 长度为 `2` 的区间只要两个字符相等就成立
- 更长区间要看首尾是否相同，以及内部区间是否已经是回文

## 公式

若 `s[i] == s[j]` 且 `(j - i <= 1 or dp[i + 1][j - 1])`，则 `dp[i][j] = True`

## 复杂度

- 时间复杂度：`O(n^2)`
- 空间复杂度：`O(n^2)`
