# 多边形三角剖分的最低得分

## 题目

给定一个凸多边形的顶点权值数组 `values`，每次选择三个顶点形成一个三角形，得分为三点权值乘积。把整个多边形剖分成若干三角形后，求总得分最小值。

## 为什么是区间 DP

如果把顶点 `i` 到 `j` 看成一段边界，选择中间某个点 `k` 作为最后一次划分，就会分成两个更小的子区间。

## 四步法

1. 定义状态：`dp[i][j]` 表示顶点区间 `[i, j]` 构成的子多边形完成三角剖分后的最低得分。
2. 写转移：枚举中间顶点 `k`，则 `dp[i][j] = min(dp[i][k] + dp[k][j] + values[i] * values[k] * values[j])`。
3. 初始化：当区间长度小于 `2` 条边时无法形成三角形，得分为 `0`。
4. 遍历顺序：按区间长度从小到大枚举。

## 递推阶梯

- 长度不足的区间不能形成三角形
- 一旦区间足够大，就尝试每个中间点作为分割点
- 每次分割都会新增一个三角形

## 公式

`dp[i][j] = min(dp[i][k] + dp[k][j] + values[i] * values[k] * values[j])`

## 复杂度

- 时间复杂度：`O(n^3)`
- 空间复杂度：`O(n^2)`
