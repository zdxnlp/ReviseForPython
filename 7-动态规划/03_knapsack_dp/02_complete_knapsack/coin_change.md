# 零钱兑换

## 题目

给定不同面额的硬币 `coins` 和总金额 `amount`，每种硬币可以无限使用，求凑成总金额所需的最少硬币数。

## 为什么是完全背包

每种硬币可以重复使用多次，所以这是“物品可重复选取”的完全背包。

## 四步法

1. 定义状态：`dp[j]` 表示凑出金额 `j` 所需的最少硬币数。
2. 写转移：使用当前硬币后，`dp[j] = min(dp[j], dp[j - coin] + 1)`。
3. 初始化：`dp[0] = 0`，其余先设为无穷大。
4. 遍历顺序：外层遍历硬币，内层容量从小到大，答案看 `dp[amount]`。

## 递推阶梯

- 金额为 `0` 时不需要硬币
- 对每个金额，尝试最后一枚硬币是什么
- 因为硬币能重复使用，所以容量要正序遍历

## 公式

`dp[j] = min(dp[j], dp[j - coin] + 1)`

## 复杂度

- 时间复杂度：`O(len(coins) * amount)`
- 空间复杂度：`O(amount)`
