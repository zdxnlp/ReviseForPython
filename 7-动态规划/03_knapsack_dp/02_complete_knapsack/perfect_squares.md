# 完全平方数

## 题目

给定整数 `n`，返回和为 `n` 的完全平方数的最少数量。

## 为什么是完全背包

平方数 `1, 4, 9, 16...` 可以重复使用，目标是凑出总和 `n` 的最少物品数。

## 四步法

1. 定义状态：`dp[j]` 表示凑出和为 `j` 的最少完全平方数个数。
2. 写转移：若当前平方数为 `square`，则 `dp[j] = min(dp[j], dp[j - square] + 1)`。
3. 初始化：`dp[0] = 0`，其余设为无穷大。
4. 遍历顺序：先枚举平方数，再从小到大枚举容量，答案是 `dp[n]`。

## 递推阶梯

- 先生成所有不超过 `n` 的平方数
- 每个平方数都能使用多次
- 每次更新都表示“最后再放入一个平方数”

## 公式

`dp[j] = min(dp[j], dp[j - square] + 1)`

## 复杂度

- 时间复杂度：`O(n * sqrt(n))`
- 空间复杂度：`O(n)`
