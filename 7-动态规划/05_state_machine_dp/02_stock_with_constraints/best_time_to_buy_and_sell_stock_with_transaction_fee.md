# 买卖股票的最佳时机含手续费

## 题目

可以多次交易，每次卖出时需要支付手续费 `fee`，求最大利润。

## 为什么是状态机 DP

限制只体现在买卖转换上，仍然是“持股 / 不持股”两种状态，但转移时要把手续费算进去。

## 四步法

1. 定义状态：
   - `dp[i][0]`：第 `i` 天结束时不持股的最大利润
   - `dp[i][1]`：第 `i` 天结束时持股的最大利润
2. 写转移：
   - `dp[i][0] = max(dp[i - 1][0], dp[i - 1][1] + prices[i] - fee)`
   - `dp[i][1] = max(dp[i - 1][1], dp[i - 1][0] - prices[i])`
3. 初始化：`dp[0][0] = 0`，`dp[0][1] = -prices[0]`
4. 遍历顺序：按天推进，答案是 `dp[n - 1][0]`。

## 递推阶梯

- 卖出时把手续费扣掉
- 其他逻辑与不限次数的股票题相同

## 公式

- `dp[i][0] = max(dp[i - 1][0], dp[i - 1][1] + prices[i] - fee)`
- `dp[i][1] = max(dp[i - 1][1], dp[i - 1][0] - prices[i])`

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`，可优化到 `O(1)`
