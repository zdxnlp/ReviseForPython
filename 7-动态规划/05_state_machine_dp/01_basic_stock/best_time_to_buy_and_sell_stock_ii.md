# 买卖股票的最佳时机 II

## 题目

给定价格数组 `prices`，你可以完成多次交易，但同一时间只能持有一股股票，求最大利润。

## 为什么是状态机 DP

每天仍然只会落在“持股”和“不持股”两种状态，只是这次允许多次买卖。

## 四步法

1. 定义状态：
   - `dp[i][0]`：第 `i` 天结束时不持股的最大利润
   - `dp[i][1]`：第 `i` 天结束时持股的最大利润
2. 写转移：
   - `dp[i][0] = max(dp[i - 1][0], dp[i - 1][1] + prices[i])`
   - `dp[i][1] = max(dp[i - 1][1], dp[i - 1][0] - prices[i])`
3. 初始化：`dp[0][0] = 0`，`dp[0][1] = -prices[0]`
4. 遍历顺序：从第 `1` 天到最后一天，答案是 `dp[n - 1][0]`。

## 递推阶梯

- 不持股：昨天就不持股，或今天卖出
- 持股：昨天就持股，或今天从不持股状态买入

## 公式

- `dp[i][0] = max(dp[i - 1][0], dp[i - 1][1] + prices[i])`
- `dp[i][1] = max(dp[i - 1][1], dp[i - 1][0] - prices[i])`

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`，可优化到 `O(1)`
