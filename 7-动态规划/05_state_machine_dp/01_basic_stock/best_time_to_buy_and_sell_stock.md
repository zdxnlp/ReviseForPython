# 买卖股票的最佳时机

## 题目

给定数组 `prices`，第 `i` 天的价格为 `prices[i]`，只允许交易一次，求最大利润。

## 为什么是状态机 DP

每天都处在有限状态里。对于这题，核心状态只有两个：手里持股、手里不持股。

## 四步法

1. 定义状态：
   - `dp[i][0]`：第 `i` 天结束时不持股的最大利润
   - `dp[i][1]`：第 `i` 天结束时持股的最大利润
2. 写转移：
   - `dp[i][0] = max(dp[i - 1][0], dp[i - 1][1] + prices[i])`
   - `dp[i][1] = max(dp[i - 1][1], -prices[i])`
3. 初始化：`dp[0][0] = 0`，`dp[0][1] = -prices[0]`
4. 遍历顺序：按天从前往后递推，答案是最后一天不持股的利润。

## 递推阶梯

- 不持股状态：昨天就不持股，或者今天把股票卖掉
- 持股状态：昨天就持股，或者今天第一次买入

## 公式

- `dp[i][0] = max(dp[i - 1][0], dp[i - 1][1] + prices[i])`
- `dp[i][1] = max(dp[i - 1][1], -prices[i])`

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`，可优化到 `O(1)`
