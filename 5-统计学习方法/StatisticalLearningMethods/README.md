# 《统计学习方法》实验与面试题库

本项目基于李航老师的《统计学习方法》（第一版及第二版），旨在为学习者提供每个章节核心算法的**代码实现**（包含从零手写实现与调用成熟库实现）以及**最新高频面试题目与标准答案**。

## 目录结构

### 监督学习部分
* `ch02_perceptron/`: 感知机
* `ch03_knn/`: K近邻法 (KNN)
* `ch04_naive_bayes/`: 朴素贝叶斯法
* `ch05_decision_tree/`: 决策树
* `ch06_logistic_regression/`: 逻辑斯谛回归与最大熵模型
* `ch07_svm/`: 支持向量机 (SVM)
* `ch08_adaboost/`: 提升方法 (AdaBoost)
* `ch09_em/`: EM算法及其推广
* `ch10_hmm/`: 隐马尔可夫模型 (HMM)

### 无监督学习部分
* `ch14_clustering/`: 聚类方法 (K-Means 等)
* `ch15_svd/`: 奇异值分解 (SVD)
* `ch16_pca/`: 主成分分析 (PCA)

## 环境配置

请使用以下命令安装所需依赖：

```bash
pip install -r requirements.txt
```

## 使用说明

- 每一个章节的文件夹中包含一个 `.py` 文件，展示了算法的**自行实现**与**调用库实现**两种方式。
- 每一个章节的文件夹中包含一个 `interview_questions.md` 文件，整理了针对该算法的最新的、常见的机器学习面试题及标准答案。

祝学习顺利！
