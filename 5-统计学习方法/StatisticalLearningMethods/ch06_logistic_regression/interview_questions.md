# 第六章：逻辑斯谛回归与最大熵模型 面试题及标准答案

## 1. 逻辑斯谛回归（Logistic Regression）的损失函数是什么？如何推导的？
**标准答案**：
逻辑回归的损失函数是**对数似然损失函数**（或称为交叉熵损失函数 Cross-Entropy Loss）。
**推导过程**：
1. 逻辑回归假设样本服从伯努利分布，预测概率为 $P(y=1|x) = p$ 和 $P(y=0|x) = 1-p$。
2. 整合为一个公式：$P(y|x) = p^y (1-p)^{1-y}$。
3. 对于所有独立同分布的训练样本，其似然函数为：$L = \prod_{i=1}^N p_i^{y_i} (1-p_i)^{1-y_i}$。
4. 取对数得到对数似然函数：$\ln L = \sum_{i=1}^N [y_i \ln p_i + (1-y_i) \ln (1-p_i)]$。
5. 我们的目标是最大化似然函数，等价于最小化负对数似然函数（即交叉熵）：$Loss = -\frac{1}{N} \sum_{i=1}^N [y_i \ln p_i + (1-y_i) \ln (1-p_i)]$。

## 2. 逻辑回归为什么要使用 Sigmoid 函数？
**标准答案**：
从数学推导上，当假设数据特征满足高斯分布，且各个类别的协方差矩阵相等时，通过贝叶斯定理推导后验概率 $P(Y=1|X)$，化简后刚好是 Sigmoid 函数的形式。
从直观上：
1. **值域**：Sigmoid 函数将连续的实数映射到 $(0, 1)$ 区间，非常适合表示概率。
2. **可导性**：Sigmoid 函数是平滑且连续可导的，且其导数计算非常简单：$f'(x) = f(x)(1 - f(x))$，这极大地方便了梯度下降法的推导和计算。
3. **最大熵原理**：逻辑回归是最大熵模型在二分类问题下的特例。Sigmoid 函数的形式是满足给定约束条件下，使熵最大（即最不带有主观偏见）的概率分布形式。

## 3. 逻辑回归与支持向量机（SVM）的区别与联系是什么？
**标准答案**：
**联系**：
1. 两者都是监督学习算法，最初都是为了解决二分类问题而设计的。
2. 在不考虑核函数的情况下，两者都是**线性分类器**，寻找一个超平面将数据分开。
3. 两者都可以加入正则化项（L1/L2）来防止过拟合。

**区别**：
1. **损失函数不同**：逻辑回归使用的是**交叉熵损失**（Log Loss），SVM使用的是**合页损失**（Hinge Loss）。
2. **对数据的敏感度不同**：逻辑回归的决策边界受所有数据点的影响（所有点都参与梯度计算）；SVM的决策边界仅由少数**支持向量**决定，对远离边界的噪声点不敏感，鲁棒性更强。
3. **输出结果**：逻辑回归输出的是分类的概率（0到1之间），而SVM直接输出硬分类标签（+1或-1）或距离。
4. **非线性问题**：SVM天生设计了核技巧（Kernel Trick），非常适合处理非线性问题；逻辑回归虽然也可以用核，但计算复杂度过高，极少使用。

## 4. 逻辑回归在训练时如果遇到数据严重不平衡怎么办？
**标准答案**：
1. **重采样（Resampling）**：对少数类进行过采样（如SMOTE算法），或对多数类进行欠采样（Random Under-sampling）。
2. **调整样本权重（Class Weight）**：在计算损失函数时，为少数类的样本赋予更大的权重，使得模型更加关注少数类的误分类情况（例如在sklearn中设置 `class_weight='balanced'`）。
3. **调整阈值**：逻辑回归默认的分类阈值是0.5，如果正负样本比例为1:10，可以根据验证集的表现，将阈值调整为0.1左右。
4. **更换评价指标**：不能使用准确率（Accuracy），必须使用精确率（Precision）、召回率（Recall）、F1-Score 或 AUC 值来评估模型。
