# 目标和

## 题目

给定整数数组 `nums` 和目标值 `target`，给每个数字添加 `+` 或 `-` 号，问有多少种表达式结果等于 `target`。

## 为什么是 0/1 背包

设加正号的和为 `P`，加负号的和为 `N`，则有：

- `P - N = target`
- `P + N = sum(nums)`

两式相加得到：`2P = target + sum(nums)`，所以问题转化为“选出若干数，和为 `P` 的方案数”，每个数仍然只能用一次。

## 四步法

1. 定义状态：`dp[j]` 表示凑出和为 `j` 的方案数。
2. 写转移：加入数字 `num` 后，`dp[j] += dp[j - num]`。
3. 初始化：`dp[0] = 1`，表示什么都不选是一种方案。
4. 遍历顺序：外层遍历数字，内层容量倒序，答案是 `dp[bag]`。

## 递推阶梯

- 先完成代数转化
- 若 `target + sum(nums)` 为负数或奇数，直接无解
- 然后做“装满容量为 `bag` 的方案数”问题

## 公式

`dp[j] = dp[j] + dp[j - num]`

## 复杂度

- 时间复杂度：`O(n * bag)`
- 空间复杂度：`O(bag)`
