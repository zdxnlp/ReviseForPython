# 最长回文子序列

## 题目

给定字符串 `s`，求其中最长回文子序列的长度。

## 为什么是区间 DP

答案与一个区间 `s[i..j]` 有关，并且更大区间的答案依赖更小区间的答案。

## 四步法

1. 定义状态：`dp[i][j]` 表示子串 `s[i..j]` 中最长回文子序列的长度。
2. 写转移：
   - 若 `s[i] == s[j]`，则 `dp[i][j] = dp[i + 1][j - 1] + 2`
   - 否则 `dp[i][j] = max(dp[i + 1][j], dp[i][j - 1])`
3. 初始化：单个字符本身就是回文，所以 `dp[i][i] = 1`。
4. 遍历顺序：因为依赖更短区间，要让 `i` 从大到小，`j` 从小到大。

## 递推阶梯

- 区间长度为 `1` 时答案为 `1`
- 区间变长后，比较左右端点是否相等
- 相等就向内缩并加 `2`
- 不等就舍弃一端取更优

## 公式

- 相等时：`dp[i][j] = dp[i + 1][j - 1] + 2`
- 不等时：`dp[i][j] = max(dp[i + 1][j], dp[i][j - 1])`

## 复杂度

- 时间复杂度：`O(n^2)`
- 空间复杂度：`O(n^2)`
