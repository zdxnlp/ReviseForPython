# 戳气球

## 题目

有一排气球，每个气球上有一个数字。戳破第 `k` 个气球可获得 `nums[left] * nums[k] * nums[right]` 枚硬币，其中 `left` 和 `right` 是 `k` 两侧尚未被戳破的气球。求能获得的最大硬币数。

## 为什么是区间 DP

当你固定“最后一个被戳破的气球”时，左右两边会变成两个互不影响的独立区间。

## 四步法

1. 定义状态：`dp[i][j]` 表示开区间 `(i, j)` 内所有气球都戳完时的最大硬币数。
2. 写转移：枚举最后一个戳破的气球 `k`，则 `dp[i][j] = max(dp[i][k] + dp[k][j] + points[i] * points[k] * points[j])`。
3. 初始化：空区间没有收益，所以默认是 `0`。
4. 遍历顺序：按区间长度从小到大枚举，保证子区间已经算好。

## 递推阶梯

- 在两端补上虚拟气球 `1`
- 把问题改成“谁最后戳”
- 最后戳 `k` 时，左右收益已经由更小区间确定

## 公式

`dp[i][j] = max(dp[i][k] + dp[k][j] + points[i] * points[k] * points[j])`

其中 `i < k < j`。

## 复杂度

- 时间复杂度：`O(n^3)`
- 空间复杂度：`O(n^2)`
