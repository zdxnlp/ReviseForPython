# 爬楼梯

## 题目

每次可以爬 `1` 阶或 `2` 阶，问爬到第 `n` 阶有多少种不同方法。

## 为什么是线性 DP

当前位置 `i` 的答案只依赖前两个位置，所以可以沿着下标从小到大递推。

## 四步法

1. 定义状态：`dp[i]` 表示到达第 `i` 阶的方法数。
2. 写转移：最后一步要么从 `i - 1` 来，要么从 `i - 2` 来，所以 `dp[i] = dp[i - 1] + dp[i - 2]`。
3. 初始化：`dp[1] = 1`，`dp[2] = 2`。
4. 遍历顺序：从 `3` 递推到 `n`，答案是 `dp[n]`。

## 递推阶梯

- 第 `1` 阶：`1` 种
- 第 `2` 阶：`2` 种
- 第 `3` 阶：`dp[3] = dp[2] + dp[1] = 3`
- 第 `4` 阶：`dp[4] = dp[3] + dp[2] = 5`

可以看到它和斐波那契型递推完全一致。

## 公式

`dp[i] = dp[i - 1] + dp[i - 2]`

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`，可优化到 `O(1)`
