# 打家劫舍

## 题目

给定数组 `nums`，每个元素表示一间房屋里的金额。相邻房屋不能同时偷，问最多能偷到多少钱。

## 为什么是线性 DP

每个位置都面对“偷 or 不偷”的线性决策，当前最优值只依赖前两个位置。

## 四步法

1. 定义状态：`dp[i]` 表示偷到下标 `i` 这间房为止，能够获得的最大金额。
2. 写转移：不偷当前房屋时取 `dp[i - 1]`，偷当前房屋时取 `dp[i - 2] + nums[i]`，所以 `dp[i] = max(dp[i - 1], dp[i - 2] + nums[i])`。
3. 初始化：`dp[0] = nums[0]`，`dp[1] = max(nums[0], nums[1])`。
4. 遍历顺序：从左到右递推，答案是 `dp[n - 1]`。

## 递推阶梯

- 第 `0` 间：只能偷它自己
- 第 `1` 间：两间里选钱更多的
- 第 `2` 间开始：每一步都比较“继承前一间”还是“隔一间再偷当前”

## 公式

`dp[i] = max(dp[i - 1], dp[i - 2] + nums[i])`

## 复杂度

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`，可优化到 `O(1)`
