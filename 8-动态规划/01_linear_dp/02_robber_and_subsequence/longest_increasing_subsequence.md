# 最长递增子序列

## 题目

给定整数数组 `nums`，返回最长严格递增子序列的长度。

## 为什么是线性 DP

虽然题目求的是子序列，但可以按下标线性推进，在每个位置汇总“以它结尾”的最优解。

## 四步法

1. 定义状态：`dp[i]` 表示以 `nums[i]` 结尾的最长递增子序列长度。
2. 写转移：若 `nums[j] < nums[i]`，则可以把 `nums[i]` 接在 `j` 的后面，所以 `dp[i] = max(dp[i], dp[j] + 1)`。
3. 初始化：每个元素单独成序列，所以 `dp[i] = 1`。
4. 遍历顺序：外层枚举 `i`，内层枚举 `0..i-1`，答案是 `max(dp)`。

## 递推阶梯

- 先假设每个位置都只能自己单独成序列
- 再去找前面比它小的数
- 找到一个更好的前驱，就更新当前长度

## 公式

`dp[i] = max(1, dp[j] + 1)`，其中 `0 <= j < i` 且 `nums[j] < nums[i]`

## 复杂度

- 时间复杂度：`O(n^2)`
- 空间复杂度：`O(n)`
