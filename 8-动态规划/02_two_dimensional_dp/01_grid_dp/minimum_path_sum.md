# 最小路径和

## 题目

给定一个非负整数网格 `grid`，从左上角走到右下角，每次只能向右或向下，求路径数字和的最小值。

## 为什么是二维 DP

每个格子的最优值由坐标决定，并且只依赖它的上方和左方两个格子。

## 四步法

1. 定义状态：`dp[i][j]` 表示走到 `(i, j)` 时的最小路径和。
2. 写转移：`dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j]`。
3. 初始化：`dp[0][0] = grid[0][0]`，第一行和第一列只能单方向累加。
4. 遍历顺序：从左上到右下，答案是 `dp[m - 1][n - 1]`。

## 递推阶梯

- 起点只有一个值
- 第一行只能从左边过来
- 第一列只能从上面过来
- 其他格子取更小的来源再加上自身权重

## 公式

`dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j]`

## 复杂度

- 时间复杂度：`O(mn)`
- 空间复杂度：`O(mn)`
