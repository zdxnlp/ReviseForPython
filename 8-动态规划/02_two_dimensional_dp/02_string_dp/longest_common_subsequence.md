# 最长公共子序列

## 题目

给定两个字符串 `text1` 和 `text2`，返回它们的最长公共子序列长度。

## 为什么是二维 DP

状态同时依赖两个字符串的前缀长度，典型地用 `i, j` 表示“双序列比较”。

## 四步法

1. 定义状态：`dp[i][j]` 表示 `text1` 前 `i` 个字符与 `text2` 前 `j` 个字符的最长公共子序列长度。
2. 写转移：
   - 若 `text1[i - 1] == text2[j - 1]`，则 `dp[i][j] = dp[i - 1][j - 1] + 1`
   - 否则 `dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])`
3. 初始化：空串与任何串的公共子序列长度为 `0`。
4. 遍历顺序：按前缀长度从小到大枚举，答案是 `dp[m][n]`。

## 递推阶梯

- 两个末尾字符相等，就把答案建立在更短前缀上
- 两个末尾字符不等，就丢掉其中一个字符继续比较

## 公式

- 相等时：`dp[i][j] = dp[i - 1][j - 1] + 1`
- 不等时：`dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])`

## 复杂度

- 时间复杂度：`O(mn)`
- 空间复杂度：`O(mn)`
