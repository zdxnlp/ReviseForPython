# 投资最大收益周期 - 动态规划解法

## 问题本质
这是经典的**最大子数组和问题**（Maximum Subarray Problem），也称为Kadane算法。

## 一、状态定义

### 方法1：以当前位置结尾的最大子数组和

**状态定义**：
- `dp[i]`：表示以第 `i` 个元素**结尾**的连续子数组的最大和

**为什么这样定义？**
- 必须包含第 `i` 个元素，保证了子数组的连续性
- 通过遍历所有位置，可以找到全局最大值

### 方法2：到当前位置的最大收益（可选）

**状态定义**：
- `dp[i]`：表示前 `i` 个元素中，连续子数组的最大和（不一定包含第 `i` 个元素）

## 二、状态转移方程

### 方法1的状态转移

对于位置 `i`，有两种选择：

1. **将 nums[i] 加入前面的子数组**：`dp[i-1] + nums[i]`
2. **从 nums[i] 重新开始**：`nums[i]`

**状态转移方程**：
```
dp[i] = max(dp[i-1] + nums[i], nums[i])
```

**简化形式**：
```
dp[i] = max(dp[i-1], 0) + nums[i]
```

**最终答案**：
```
result = max(dp[0], dp[1], ..., dp[n-1])
```

## 三、求解步骤

### 步骤1：初始化
```python
dp[0] = nums[0]  # 第一个元素结尾的最大和就是它本身
result = dp[0]   # 记录全局最大值
```

### 步骤2：状态转移
```python
for i in range(1, n):
    # 选择：加入前面 or 重新开始
    dp[i] = max(dp[i-1] + nums[i], nums[i])
    # 更新全局最大值
    result = max(result, dp[i])
```

### 步骤3：返回结果
```python
return result
```

## 四、完整代码实现

### 标准DP版本（使用数组）
```python
def solve_dp():
    n = int(input())
    if n <= 0:
        return 0
    
    nums = list(map(int, input().split()))
    if len(nums) != n:
        return 0
    
    # dp[i] 表示以 nums[i] 结尾的最大子数组和
    dp = [0] * n
    dp[0] = nums[0]
    result = dp[0]
    
    for i in range(1, n):
        # 状态转移：加入前面 or 重新开始
        dp[i] = max(dp[i-1] + nums[i], nums[i])
        # 更新全局最大值
        result = max(result, dp[i])
    
    return result
```

### 空间优化版本（滚动变量）
```python
def solve_optimized():
    n = int(input())
    if n <= 0:
        return 0
    
    nums = list(map(int, input().split()))
    if len(nums) != n:
        return 0
    
    # 只需要记录前一个状态
    prev_max = nums[0]  # dp[i-1]
    result = nums[0]
    
    for i in range(1, n):
        # 状态转移
        current_max = max(prev_max + nums[i], nums[i])
        # 更新结果
        result = max(result, current_max)
        # 滚动更新
        prev_max = current_max
    
    return result
```

## 五、复杂度分析

- **时间复杂度**：O(n) - 只需遍历一次数组
- **空间复杂度**：
  - 标准DP版本：O(n) - 需要dp数组
  - 优化版本：O(1) - 只需常数个变量

## 六、示例演示

### 输入
```
6
-2 1 -3 4 -1 2
```

### DP过程

| i | nums[i] | dp[i-1] | dp[i] = max(dp[i-1]+nums[i], nums[i]) | result |
|---|---------|---------|----------------------------------------|--------|
| 0 | -2      | -       | -2                                     | -2     |
| 1 | 1       | -2      | max(-2+1, 1) = 1                       | 1      |
| 2 | -3      | 1       | max(1-3, -3) = -2                      | 1      |
| 3 | 4       | -2      | max(-2+4, 4) = 4                       | 4      |
| 4 | -1      | 4       | max(4-1, -1) = 3                       | 4      |
| 5 | 2       | 3       | max(3+2, 2) = 5                        | 5      |

**最大子数组**：[4, -1, 2]，和为 5

## 七、关键理解

1. **状态定义的关键**：必须以当前元素结尾，保证连续性
2. **决策的本质**：每个位置都在问"要不要和前面的连在一起？"
3. **贪心的思想**：如果前面的和是负数，不如重新开始
4. **全局最优**：通过记录每个位置的局部最优，找到全局最优
